# GuruSaarthi Docs

This repository contains the **Privacy Policy** and **Terms & Conditions** for GuruSaarthi, an AI conversational bot designed to provide emotional guidance and reflective support.

The HTML files in this repository are intended to be hosted via GitHub Pages, providing official URLs that can be linked in the GuruSaarthi bot interface on BharatGPT.

## Files

- `privacy.html` — GuruSaarthi Privacy Policy
- `terms.html` — GuruSaarthi Terms & Conditions

## Purpose

These documents ensure transparency and compliance for users of GuruSaarthi. They outline data usage, safety considerations, and limitations of the AI bot.
